Tanya Jawab

  1. Apa itu adaptive dan responsive web design? Apakah kamu setuju bahwa teknik tersebut bagus? Mengapa atau mengapa tidak?

    Adaptive: Web design yang di load hanya untuk screen size yang terdetect, dan harus ada adaptive design untuk setiap screen size yang diprediksi (320, 480, 760, 960, 1200, 1600, dll). Responsive: Web design yang sama digunakan untuk screen size yang berbeda, tetapi tampilannya responsif dengan screen size yang terdetect.

    Responsive lebih mudah dicoding, karena tidak butuh terlalu banyak design. Adaptive loadnya jauh lebih cepat, karena kode yang tidak dibutuhkan tidak di load.
  2. Apakah hadirnya smartphone dan tablet mengubah desain web secara signifikan? Bagaimana kira-kira teknologi baru akan mengubah desain web di masa depan?

    Ya, karena web designer harus mengetahui jika User paling sering mengunjung website lewat tablet, handphone, atau laptop, dan juga dengan OS yang mana. Teknologi baru akan membuat web design menjadi jauh lebih mudah, dan responsive web design untuk load jauh lebih cepat.
  3. Bagaimana kira-kira orang yang memiliki kekurangan fisik menggunakan website? Apakah kita perlu menyesuaikan website untuk mereka juga? Solusi apa yang dapat mengatasi keadaan tersebut?

    Ya, jika mayoritas pengguna website tersebut memiliki kekurangan fisik. Salah satu solusinya adalah voice command.
  4. Ide atau konsep apa yang sesuai denganmu saat mempelajari desain web?

    Konsep responsive web design.
